## 中文翻译计划

本书是 Edward.O.Wilson 教授的代表作，曾获普利策奖。兴趣使然，就了解到本书，可惜目前暂无中文版本。与其苦等别人来译，不如自己动手，以求积水成渊。希望更多的人能看到并了解蚂蚁，放下点人类的傲慢，对生命多一点尊重。

本人非专业人士，翻译水平有限，多有不当之处，还请指正。
